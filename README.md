# parse-mail

## Usage

```
Usage: parse-mail [OPTIONS]

Options:
  -D PATH
  -i, --input PATH
  -d, --depth INT
  -o, --output PATH
  -P, --parallel-file-read, --parallel INT
  -p, --print                      Also print the result email summary
  -h, --help                       Show this message and exit
```

For example:

```bash
❯ parse-mail -i ./msg 
09-12T01:19:02 - I [Main] >>> Successfully write result csv to ./result.csv 
```
